avx::Int256¶

class Int256¶

Vector containing 8 signed 32-bit integers.

Public Types

using storedType = int¶

Public Functions

inline Int256()¶

Default constructor. Initializes vector with zeros.

inline Int256(const int *init)¶

Initializes vector by loading data from memory (via _mm256_lddq_si256).

Parameters:

init – Valid memory addres of minimal size of 256-bits (32 bytes).

inline Int256(const int &init)¶

Initializes vector with const value. Each cell will be set with value of init.

Parameters:

init – Value to be set.

inline void load(const int *sP)¶

Loads data from memory into vector (memory should be of size of at least 32 bytes). Memory doesn’t need to be aligned to any specific boundary. If sP is nullptr this method has no effect.

Parameters:

sP – Pointer to memory from which to load data.

inline void save(std::array<int, 8> &dest) const¶

Saves vector data into an array.

Parameters:

dest – Destination array.

inline void save(int *dest) const¶

Saves data into given memory address. Memory doesn’t need to be aligned to any specific boundary.

Parameters:

dest – A valid (non-nullptr) memory address with size of at least 32 bytes.

inline void saveAligned(int *dest) const¶

Saves data from vector into given memory address. Memory needs to be aligned on 32 byte boundary. See https://www.intel.com/content/www/us/en/docs/intrinsics-guide/index.html for more details.

Parameters:

dest – A valid (non-NULL) memory address aligned to 32-byte boundary.
