Loads data from memory into vector (memory should be of size of at least 32 bytes). Memory doesn’t need to be aligned to any specific boundary. If sP is nullptr this method has no effect.

Parameters:

sP – Pointer to memory from which to load data.

inline void save(unsigned long long *dest) const¶

Saves data into given memory address. Memory doesn’t need to be aligned to any specific boundary.

Parameters:

dest – A valid (non-nullptr) memory address with size of at least 32 bytes.

inline void saveAligned(unsigned long long *dest) const¶

Saves data from vector into given memory address. Memory needs to be aligned on 32 byte boundary. See https://www.intel.com/content/www/us/en/docs/intrinsics-guide/index.html for more details.

Parameters:

dest – A valid (non-NULL) memory address aligned to 32-byte boundary.
