avx::Short256¶

class Short256¶

Class providing vectorized version of short. Can hold 16 individual short values. Provides arithmetic and bitwise operators. Provides comparison operators == !=.

Public Types

Type that is stored inside vector.

inline void load(const short *pSrc)¶

Loads data from memory into vector (memory should be of size of at least 32 bytes). Memory doesn’t need to be aligned to any specific boundary. If sP is nullptr this method has no effect.

Parameters:

Throws:

std::invalid_argument – If in Debug mode and pSrc is nullptr. In Release mode no checks are performed to improve efficiency.

inline void save(short *pDest) const¶

See https://en.cppreference.com/w/cpp/memory/c/aligned_alloc for more details.

inline void saveAligned(short *pDest) const¶

Saves data to destination in memory. The memory must be aligned at 32-byte boundary.
