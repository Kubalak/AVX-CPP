Supported operators
===================

+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| Operator     | Supported type(s)        | Instructions used              | Comment                                                    |
+==============+==========================+================================+============================================================+
| +, +=        |``char``, ``avx::Char256``|  AVX, AVX2                     | AVX is additionally used when argument is scalar.          |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| -, -=        |``char``, ``avx::Char256``|  AVX, AVX2                     | Same as above.                                             |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| \*, \*=      |``char``, ``avx::Char256``| AVX, AVX2 or AVX512F, AVX512BW |  AVX512F/BW is used if ``BUILD_USE_AVX512`` is set.        |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| /, /=        |``char``, ``avx::Char256``| Same as above + AVX512VL       |  As above + same for AVX512VL                              |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| %, %=        |``char``, ``avx::Char256``| Same as above                  |  Same as above                                             |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| <<, <<=      |``char``, ``avx::Char256``| AVX2, AVX512BW                 |  AVX512BW is used if ``BUILD_USE_AVX512`` is set.          |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| >>, >>=      |``char``, ``avx::Char256``| AVX2, AVX512BW                 |  AVX512BW is used if ``BUILD_USE_AVX512`` is set.          |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| &, &=        |``char``, ``avx::Char256``| AVX, AVX2                      |  AVX is additionally used when argument is scalar.         |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| \|, \|=      |``char``, ``avx::Char256``| AVX, AVX2                      |  AVX is additionally used when argument is scalar.         |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| ^, ^=        |``char``, ``avx::Char256``| AVX, AVX2                      |  AVX is additionally used when argument is scalar.         |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| ~            | \-                       | AVX2                           | Unary operator.                                            |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| ``[]``       | ``unsigned int``         | N/A                            | In GCC and MSVC disassembly no SIMD instructions were used |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| ==           |``char``, ``avx::Char256``| AVX, AVX2                      | Returns true if ALL values ARE equal.                      |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
| !=           |``char``, ``avx::Char256``| AVX, AVX2                      | Returns true if ANY value is NOT equal.                    |
+--------------+--------------------------+--------------------------------+------------------------------------------------------------+
